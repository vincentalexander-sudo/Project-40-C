# 40 Projects Challenge - C Programming

Repository ini berisi kumpulan proyek pemrograman C sebagai bagian dari tantangan 40 proyek.

## 📅 Progres Harian

### Day 1: 
1. **Data Range**: Menghitung jangkauan data (terbesar - terkecil) dan nilai rata-rata data.
2. **Depresiasi Tahunan**: Menghitung penyusutan nilai aset.
3. **Perhitungan Kotak**: Menghitung permukaan kotak.
4. **Perhitungan Resistensi**: Menghitung hambatan listrik.
5. **Tagihan Air**: Simulasi perhitungan biaya pemakaian air.

### Day 2:
6. **Faktorial**: Menghitung hasil perkalian berurutan (n!).
7. **Pengecekan Bilangan Prima**: Menentukan apakah angka adalah bilangan prima.
8. **Kelipatan Bilangan**: Menampilkan deret angka kelipatan.
9. **Perpangkatan**: Menghitung hasil x pangkat y.
10. **Permutasi**: Menghitung susunan objek (P(n, r)).

### Day 3: 
11. **Binomial**: Menghitung koefisien binomial (kombinasi).
12. **Deret Aritmatika**: Membuat barisan angka berdasarkan selisih kustom.
13. **Konversi Suhu**: Mengubah Celcius ke Fahrenheit, Reamur, dan Kelvin.
14. **Perhitungan Lingkaran**: Menghitung luas dan keliling lingkaran.
15. **Ganjil Genap**: Menentukan kategori angka menggunakan operator modulo.


### Day 4: 
16. **Jajargenjang**: Menghitung luas dan keliling jajargenjang.
17. **Trapesium**: Perhitungan luas dan keliling trapesium sama kaki menggunakan input sisi miring.
18. **Belah Ketupat**: Menghitung luas (diagonal) dan keliling (sisi) dalam satu program.
19. **Segitiga Heron**: Menghitung keliling dan luas segitiga heron berdasarkan panjang tiga sisi 
20. **Energi Fisika**: Kalkulator Energi Potensial dan Energi Kinetik dengan konstanta gravitasi 9.8.
