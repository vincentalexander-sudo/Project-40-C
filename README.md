Repository ini berisi kumpulan proyek pemrograman C sebagai bagian dari tantangan 40 proyek.

## 📅 Progres Harian

### Day 1: Dasar Logika & Perhitungan
1. **Data Range**: Menghitung jangkauan data (terbesar - terkecil) dan nilai rata-rata data.
2. **Depresiasi Tahunan**: Menghitung penyusutan nilai aset.
3. **Perhitungan Kotak**: Menghitung permukaan kotak.
4. **Perhitungan Resistensi**: Menghitung hambatan listrik.
5. **Tagihan Air**: Simulasi perhitungan biaya pemakaian air.

### Day 2: Matematika & Algoritma
6. **Faktorial**: Menghitung faktorial.
7. **Pengecekan Bilangan Prima**: Menentukan apakah angka adalah bilangan prima.
8. **Kelipatan Bilangan**: Menampilkan deret angka kelipatan.
9. **Perpangkatan**: Menghitung hasil x pangkat y.
10. **Permutasi**: Menghitung susunan objek (P(n, r)).
